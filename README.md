<h2 style="text-align: left">Objetivo</h2>
<div>Demonstrar minhas habilidades técnicas para realizar atividades <p>
de normalização de dados.</p></div>
<h2>Descrição do Projeto</h2>
<div>Para realizar as atividades, utilizei frequentemente a estrutura<p>
de try-with-resources para fechar os objetos utilizados e tratar </p>
possíveis problemas durante a execução do programa.</div>
<div>Além disso, realizei testes individuais durante a escrita do <p>
programa.</p></div>
<h3>Trade-off técnico</h3>
<div>Trade-off (1.2): Eu processei o arquivos incrementalmente, pois <p>
se o arquivo não possuísse dados de despesas, este não precisaria ser processado.</p></div>
<div>Deste modo, não ocuparia a memória desnecessariamente.</div>
<h2>Execução</h2>
<div>Escolha uma IDE da sua preferência e execute o arquivo Main.java. <p>
Caminho: Teste_de_Entrada/src/Main/</p></div>
<div>Ou abra o seu terminal, siga até o caminho acima e execute o <p>
seguinte comando: javac Main.java</p></div>
<div>Depois este: java Main</div>
