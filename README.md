<h2 style="text-align: left">Objetivo</h2>
<div>Demonstrar minhas habilidades técnicas para realizar atividades
de normalização de dados.</div>
<h2>Descrição do Projeto</h2>
<div>Para realizar as atividades, utilizei frequentemente a estrutura de try-with-resources para fechar <p>
os objetos utilizados e tratar possíveis problemas durante a execução do programa. </p></div>
<div>Além disso, realizei testes individuais durante a escrita do programa.</div>
<h3>Trade-off técnico</h3>
<div>Trade-off (1.2): Eu processei o arquivos incrementalmente, pois se o arquivo não possuísse dados de <p>
despesas, este não precisaria ser processado.</p> Deste modo, não ocuparia a memória desnecessariamente.</div>
<h2>Execução</h2>
<div>Escolha uma IDE da sua preferência e execute o arquivo Main.java. <p>
Caminho: Teste_de_Entrada/src/Main/</p></div>
<div>Ou abra o seu terminal, siga até o caminho acima e digite o seguinte comando: javac Main.java <p>
Depois, execute este: java Main</p></div>
